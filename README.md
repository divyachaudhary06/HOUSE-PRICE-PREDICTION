# HOUSE-PRICE-PREDICTION

## Project Overview

This project predicts house prices using Multiple Linear Regression. The dataset contains various housing features such as living area, number of bedrooms, bathrooms, grade, and other attributes that influence house prices.

The objective is to build a regression model that can accurately estimate house prices based on these features.

---

## Dataset

The dataset contains:

- House features (independent variables)
- House price (target variable)

Target Variable:
- Price

---

## Visualizations

### Correlation Heatmap
Used to identify relationships between features and house prices.

### Scatter Plot
Analyzed the relationship between living area and house price.

### Residual Plot
Evaluated prediction errors.

### Actual vs Predicted Plot
Compared actual house prices with model predictions.

## Model Performance

| Metric | Value |
|----------|----------|
| MAE | 105,878 |
| R² Score | 0.747 |

The model explains approximately 74.7% of the variance in house prices.

---

## Results

The Multiple Linear Regression model achieved reasonable predictive performance and successfully captured the relationship between housing features and prices.
